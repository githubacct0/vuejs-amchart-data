# vue 2 template app

## Features already configured:
* vuetify (dark theme)
* left navigation drawer
* routing - for multiple pages
* vuex - state managment

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
